# Task 02 - Use GitHub Copilot to update .NET versions

## Introduction

Zava has upgraded the Zava Storefront to .NET 8 LTS. This exercise demonstrates how Copilot can be assigned to implement issues and perform code updates automatically.

This task shows how you would assign Copilot to implement an issue and perform upgrades or improvements. 

## Description

In this task you learn how to assign Copilot to do the work to implement issues automatically.

## Success Criteria

- You understand how to assign Copilot to implement issues (Note: The solution is already updated to .NET 8 LTS)

## Key Tasks

### 01: Assign Copilot to the .NET Upgrade issue

After selecting the new issue created by Copilot, you will assign Copilot to implement the issue and create a pull request with the changes. 

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Select the issue in GitHub online as you did in the previous exercise. 

2. Assign Copilot to the issue. 

    ![Assign copilot to an issue](../../media/0502_assign_copilot.png)
    ![Assign copilot dialog](../../media/0502_assign_copilot2.png)

3. Copilot will start analyzing the solution for the changes necessary to update the solution code to .NET LTS. You can select 'View session' to watch this work in progress. 

    ![View session button](../../media/0502_viewsession.png)
    ![View session in progress](../../media/0502_viewsession2.png)

    > **NOTE** It make take 5-10 minutes for the upgrade pull request to be created. 

</details>

### 02: Review the pull request created by Copilot

After Copilot completes the implementation of the issue, you can review the pull request created by Copilot, and accept the merge. 

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Once the pull request is ready you can view it and select 'Add your review'.

    ![View pull request](../../media/0502_viewpr.png)
    ![Add your review to pull request](../../media/0502_addreview.png)

2. Merge the pull request. Select 'Merge pull request' and then 'Confirm merge'.

    ![Merge pull request](../../media/0502_mergepr.png)
    ![Confirm merge](../../media/0502_confirmmerge.png)

</details>

## Summary

You've completed this task. You now understand how Copilot can be assigned to implement issues, including building the solution and executing GitHub actions for code and quality reviews. The solution has been upgraded to .NET 8 LTS.
